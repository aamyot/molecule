Getting started

Build yourself using Gradle or Buildr, or simply download from Maven Central:

<dependency>
      <groupId>com.vtence.molecule</groupId>
      <artifactId>molecule</artifactId>
      <version>0.4-SNAPSHOT</version>
</dependency>

To use the default web server, you need to add Simple as a dependency:

<dependency>
      <groupId>org.simpleframework</groupId>
      <artifactId>simple</artifactId>
      <version>5.1.6</version>
</dependency>

Then you're ready to go:

public class HelloWorld {
    public static void main(String[] args) throws IOException {
        WebServer server = WebServer.create();
        server.run(new Application() {
            public void handle(Request request, Response response) throws Exception {
                response.body("Hello, World");
            }
        });
    }
}

Your application is now available at: http://localhost:8080
